# Hudaa-Ya
Nama project ini, Hudaa-Ya, diambil dari penggalan Surah Al-Baqarah ayat 38. فَمَنْ تَبِعَ هُدَايَ فَلَا خَوْفٌ عَلَيْهِمْ وَلَا هُمْ يَحْزَنُوْنَ. Artinya "... siapa saja yang mengikuti petunjuk-Ku tidak ada rasa takut yang menimpa mereka dan mereka pun tidak bersedih hati.”. Diharapkan dengan adanya Website ini, selain menjadi Platform pembelajaran, bisa juga menjadi petunjuk bagi siapapun yang membacanya
